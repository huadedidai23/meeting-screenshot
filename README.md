# 会议截图工具 v0.1

自动监控屏幕变化并保存截图的桌面工具，适用于会议记录、演示文稿录制等场景。

## 功能特性

- ✅ **智能截图** - 检测屏幕变化超过阈值时自动保存
- ✅ **多种模式** - 支持全屏截图和区域框选
- ✅ **灵活配置** - 可调节变化阈值（1%-20%）和截图间隔（1.0-5.0秒）
- ✅ **系统托盘** - 启动后缩小到托盘，不干扰工作
- ✅ **实时监控** - 显示已截图数量，支持暂停/继续
- ✅ **单文件 exe** - 无需安装，开箱即用

## 快速开始

### 直接使用（推荐）

1. 下载 `dist/会议截图工具_v0.1.exe`
2. 双击运行
3. 配置保存位置和参数
4. 点击"开始记录"

### 从源码运行

**环境要求：**
- Python 3.11+
- Windows 10/11

**安装依赖：**
```bash
pip install PyQt5 mss Pillow numpy pyinstaller
```

**运行：**
```bash
python -m meeting_screenshot.main
```

**打包 exe：**
```bash
pyinstaller --onefile --windowed --name "会议截图工具_v0.1" meeting_screenshot/main.py
```

## 使用说明

### 主窗口配置

1. **保存位置** - 选择截图保存文件夹（默认：桌面/会议截图_日期时间）
2. **截图模式**
   - 全屏截图：捕获整个屏幕
   - 框选区域：拖拽鼠标框选特定区域
3. **变化阈值** - 画面变化超过此百分比才保存（默认 5%）
4. **截图间隔** - 检测间隔时间（默认 1.5 秒）

### 系统托盘操作

右键托盘图标可以：
- 暂停/继续监控
- 查看已截图数量
- 重新配置参数
- 打开截图文件夹
- 退出程序

### 区域框选

1. 选择"框选区域"模式
2. 点击"重新框选"
3. 拖拽鼠标框选区域
4. 按 Enter 确认，ESC 取消

## 技术栈

- **GUI 框架**: PyQt5
- **截图库**: mss（高性能跨平台截图）
- **图像处理**: Pillow + numpy
- **打包工具**: PyInstaller
